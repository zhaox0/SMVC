Switchable Modality Clue-based Voice Conversion with Normalizing Flow

Code

Abstract

Traditional voice conversion (VC) converts source speech into target one based on the reference voice of the target speaker, while face-based VC does it on the reference face image of that. In this work, we propose Switchable Modality Clue-based Voice Conversion (SMVC), a clue-based VC model with switchable modality, which can infer converted speech optionally based on either face images or voices of the target speaker. We adopt a normalizing flow to map voice-related features extracted from face images to those from speech, and introduce a cross-modal consistency loss to align the feature distributions of face images and that of speech. Experimental results demonstrate that SMVC performs well using reference clues from either modality, whether face images or voices. Moreover, SMVC achieves the optimal performance among the face-based VC methods.
