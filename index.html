Switchable Modality Clue-based Voice Conversion with Normalizing Flow

Code

Abstract

Traditional Voice conversion (VC) converts a source speech into a target one based on a reference voice of target speaker, while face-based VC does it on a reference face image of that. In this work, we propose Switchable Modality Clue-based Voice Conversion (SMVC), a clue-based VC model with switchable modality, which can infers a converted speech optionally based on a face image or voice of target speaker.We adopt a normalizing flow to map the voice-related features extracted from a face image to that extracted from a speech, and introduce a cross-modal consistency loss to align the feature distributions from a face image and that from a speech with each other.
